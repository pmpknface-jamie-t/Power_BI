# Power BI Dashboards
### Jamie Tarquini | pmpknface

A collection of Power BI dashboards built for sports analytics, 
comic collecting, and personal projects.

## Projects

### Moneyball: Efficiency & ROI Dashboard
Analyzes MLB franchise efficiency — measuring cost per win, payroll 
index, and operating margin across eras and teams. Built using 
Retrosheet historical data with dynamic filtering by era, franchise, 
and win tier.

### LB Cole Census Analysis Dashboard
A personal analytics dashboard measuring my LB Cole comic collection 
against the broader CGC census. Built using a custom Python web scraper 
to pull live census data directly from the CGC website, combined with 
personal collection data. Features a custom Sleeper Score metric to 
identify undervalued books, a dumbbell plot showing upgrade potential, 
and dynamic filtering by genre and label type.

### LB Cole Sleeper Book & Population Dashboard
A companion dashboard analyzing population distribution, grade scarcity, 
and collecting opportunity across the broader LB Cole CGC census.
